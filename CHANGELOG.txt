- index.js, api.js 파일 생성
- vuex로부터 store import
- openweatherapi import

## Store
- `index.js`와 `api.js` 파일 생성.
- Vuex로부터 `store` import.
- `openWeatherApi` import.
- `api.js`에 `state`, `getter`, `mutations`, `actions` 각각 정의.
- `created()` : `index.vue`에서 `dispatch` 사용해 호출, `openWeatherApi`는 모듈화되어 있으므로 경로에 추가 언급.
- 임의의 좌표를 payload로 전달하고, state 접근 (객체 구조 분해).
- `store` import로 `this` 대신 `store` 사용 가능.
- `cityName()` 메서드 추가, `currentData()` 추가.
- `WeatherBox`에서 `props`로 data 정의하고, template의 `{{}}`를 data로 데이터 바인딩 업데이트.
- `Graph`에서 `graphData`를 `computed`로 이동, `hourlyData()` 추가.
- `DailyWeather` template에서 데이터 바인딩.
- `api.js`에 `namespaced` 추가.
- `index.vue`의 `async created` 부분과 `hourly data`, `import axios` 삭제.
- `api.js`에서 data, current, hourly 등을 설정.
- `index.vue` data의 axis가 `created`의 `this.axis -> dispatch -> api.js`의 `async` payload로 이어짐. 여러 state 메서드 추가.
